# [StarkEngine] 通用设置系统 (通用设置系统.md)

> **AI 检索元数据**  
> **项目名称**: StarkEngine (wing_Stark)  
> **核心领域**: StarkSetting, AddConfig, 自动化 UI 配置, 双端同步, 管理员权限控制  
> **主要目标**: 提供一套与存储系统深度绑定的、声明式的 UI 配置方案，支持开关、滑动条、下拉框及自定义按钮等多种控件。

---

## 1. 概述 (Overview)

通用设置系统（StarkSetting）是 SE 提供的 UI 交互增强模块。它允许开发者通过简单的代码注册，在游戏的暂停页面自动生成功能完备的配置面板。

### 核心特性
*   **类型丰富**: 支持布尔（开关）、数字（滑动条）、文本（输入框）、枚举（下拉框）、按钮、提示文本。
*   **权限管理**: 支持 `requireOp` 限制，仅管理员可编辑核心配置。
*   **双端支持**: 可配置服务端全局同步项或客户端本地表现项。
*   **数据绑定**: 完美契合 `@DataStorage.AutoSave` 类，实现“配置即存档”。

---

## 2. 核心类：`StarkSetting`

用于定义一个设置对象，通常作为项目的全局单例。

### 2.1 构造参数
*   **enteranceText**: 暂停页面按钮显示的文字。
*   **visible**: 是否自动在暂停页面注册按钮入口（默认为 `True`）。

---

## 3. 成员方法与配置注册

### 3.1 分页管理 (`AddCategory`)
用于将配置项归类到不同的标签页中。
*   `text`: 分页标题。
*   `priority`: 显示优先度（数值越小越靠前）。
*   `icon`: 分页图标路径。

### 3.2 配置项管理 (`AddConfig`)
核心注册方法。
*   `text`: 显示给玩家的文本。
*   `category`: 所属分页标题。
*   `clsPath`: 变量所在类的完整路径（建议使用反射获取）。
*   `varName`: 绑定的变量名。
*   `isServer`: 是否为服务端配置（默认 `False`）。
*   `requireOp`: 是否仅管理员可编辑（默认 `False`）。
*   `tips`: 鼠标悬浮显示的提示描述。
*   **data (额外数据字典)**:
    *   `enums`: `list` 枚举列表（下拉框专用）。
    *   `min` / `max`: 数字范围（滑动条专用）。
    *   `step`: 步长。

### 3.3 特殊类型判定 (String 变量)
*   **提示类型**: 变量值以 `tips:` 开头时，渲染为静态文本。
*   **按钮类型**: 变量值以 `func:` 开头时，渲染为按钮。`func:` 后的内容为 `callCode`。

---

## 4. 响应与控制接口

| 方法 | 描述 | 参数 |
| :--- | :--- | :--- |
| **RegisterCallbackOnChange** | 当绑定的配置变量改变时触发回调。 | `clsPath`, `varName`, `callback` |
| **BindButtonCall** | 绑定配置按钮的点击事件。 | `callCode`, `callback` |
| **OpenSettingPanel** | 【仅客户端】直接呼出设置面板。 | `enteranceText` |

---

## 5. 实战整合示例

```python
from ....StarkEngine.StarkServer import StarkDataStorage, StarkSetting

# 1. 初始化设置对象
SettingObj = StarkSetting('旅途 | 模块配置')

# 2. 定义数据存储类
@StarkDataStorage.AutoSave(1, uid="RuntimeConfig")
class MyConfig:
    Difficulty = '简单'
    MobLimit = 100
    EnableWeather = True
    ResetAction = 'func:call_reset' # 按钮

# 获取类路径
ConfigPath = MyConfig.__module__ + '.' + MyConfig.__name__

# 3. 注册分页
SettingObj.AddCategory('核心设置', icon='textures/ui/gear')

# 4. 注册具体配置
# 下拉框示例
SettingObj.AddConfig('游戏难度', '核心设置', ConfigPath, 'Difficulty', 
                     isServer=True, requireOp=True, 
                     data={'enums': ['简单', '硬核']})

# 滑动条示例
SettingObj.AddConfig('生物上限', '核心设置', ConfigPath, 'MobLimit', 
                     data={'min': 10, 'max': 200, 'step': 1})

# 5. 绑定按钮事件
def onReset(args):
    print("触发了重置按钮")

SettingObj.BindButtonCall('call_reset', onReset)
```
