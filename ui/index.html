  1. 确认当前活跃项目与项目路径。
    2. 
                            
  2. 实现前在官方 SDK 工具中检索资料。
    3. 
                            
  3. 生成模板或审查代码，复制结果到实际项目。
    4. 
                            
  4. 开发过程保留项目记录，避免口头结论丢失。
    5. 
                        

                    

                    

                        

                            

                                
固化流程

                                
管理员维护长期记忆的路径。

                            

                        

                        
    
                            
  1. 查看项目记录，排除不可信或未解决问题。
    2. 
                            
  2. 在记忆审查中批准可复用模块。
    3. 
                            
  3. 进入知识库确认文件已落盘。
    4. 
                            
  4. 对 Wing 执行打包，让检索命中真实项目经验。
